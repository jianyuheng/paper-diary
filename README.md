# vertification

LFW和MegaFace均为人脸识别领域重要的评测数据集。LFW是针对早期人脸验证(Face Verification)任务提出评测方法与指标，结果有借鉴意义，但已不代表目前的最难问题。
MegaFace提出的关于百万级别的1：N人脸辨识(Face Identification)任务的评测指标，难度更大，是目前学术界的新主流。
